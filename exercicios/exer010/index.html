Usando links

Links Externos

Esse link leva ao meu github, repare que esse link abre a página em outra aba do navegador, diferente deste meu youtube, isso ocorre por causa do uso das tags target e rel.
Quando se está lidando com links externos rrecomenda-se usá-los com os parametros target="_blank" rel="external".

Links internos

Para acessar a pag2, basta clicar no link.


Caso queira abrir uma página cujo arquivo está dentro de uma pasta que está no mesmo lugar do arquivo, se deve colocar o nome da pasta e o nome do arquivo.
Dessa forma: exem/pag3
